lifecycle-helper:execution-plan

Full name:

it.serendigity.maven.plugins:lifecycle-helper-maven-plugin:0.2.0-SNAPSHOT:execution-plan

Description:

List execution plan for the current project.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: validate.

Optional Parameters

Name Type Since Description
<paramFilterPlugins> String[] - Allows you to filter the specified plugins (plugin-artifact-id).
User property is: lifecycle-helper.filter.plugins.
<paramGroupBy> boolean - If enabled allows you to group execution plan result by the orderBy param, any value of MavenExecutionAttribute. It's valid only for table text output result. Order by MavenExecutionAttribute.PLAN_ORDER is not grouped by.
Default value is: false.
User property is: lifecycle-helper.groupby.
<paramLifecyclePhases> String[] - Allows you to specify which phases will be used to calculate the execution plan.
Default value is: post-clean,deploy,site-deploy.
User property is: lifecycle-helper.phases.
<paramOrderBy> MavenExecutionAttribute - Allows you to group the execution plan result by any value of MavenExecutionAttribute .
Default value is: PLAN_ORDER.
User property is: lifecycle-helper.orderby.

Parameter Details

<paramFilterPlugins>

Allows you to filter the specified plugins (plugin-artifact-id).
  • Type: java.lang.String[]
  • Required: No
  • User Property: lifecycle-helper.filter.plugins

<paramGroupBy>

If enabled allows you to group execution plan result by the orderBy param, any value of MavenExecutionAttribute. It's valid only for table text output result. Order by MavenExecutionAttribute.PLAN_ORDER is not grouped by.
  • Type: boolean
  • Required: No
  • User Property: lifecycle-helper.groupby
  • Default: false

<paramLifecyclePhases>

Allows you to specify which phases will be used to calculate the execution plan.
  • Type: java.lang.String[]
  • Required: No
  • User Property: lifecycle-helper.phases
  • Default: post-clean,deploy,site-deploy

<paramOrderBy>

Allows you to group the execution plan result by any value of MavenExecutionAttribute .
  • Type: it.serendigity.maven.plugin.lifecycle.helper.vo.MavenExecutionAttribute
  • Required: No
  • User Property: lifecycle-helper.orderby
  • Default: PLAN_ORDER
