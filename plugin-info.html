Plugin Documentation

Goals available for this plugin:

Goal Description
lifecycle-helper:execution-plan List execution plan for the current project.
lifecycle-helper:execution-plan-file List execution plan for the current project to an output file.
lifecycle-helper:help Display help information on lifecycle-helper-maven-plugin.
Call mvn lifecycle-helper:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.6
JDK 17

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>it.serendigity.maven.plugins</groupId>
          <artifactId>lifecycle-helper-maven-plugin</artifactId>
          <version>1.2.1</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>it.serendigity.maven.plugins</groupId>
        <artifactId>lifecycle-helper-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
